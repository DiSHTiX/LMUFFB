LMUFFB - Le Mans Ultimate Force Feedback
========================================

Version: 0.2.0

Description
-----------
LMUFFB provides enhanced Force Feedback for Le Mans Ultimate by reading telemetry directly 
from the game engine and calculating forces based on tire grip, suspension load, and G-forces.

Features
--------
*   Real-time FFB calculation (400Hz).
*   Grip Modulation (Understeer feel).
*   Seat of Pants (Oversteer/Yaw feel).
*   Road and Slide Texture effects.
*   DirectInput support for native wheel FFB.
