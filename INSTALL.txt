Installation Instructions for LMUFFB
====================================

1. Install vJoy
---------------
Download and install vJoy 2.1.9.1 (by jshafer817) from:
https://github.com/jshafer817/vJoy/releases

2. Install Shared Memory Plugin
-------------------------------
Copy the file `rFactor2SharedMemoryMapPlugin64.dll` (found in the 'plugins' folder of this zip) 
to your Le Mans Ultimate game directory:
`Le Mans Ultimate\Plugins\`

3. Run LMUFFB
-------------
Ensure `vJoyInterface.dll` is in the same folder as `LMUFFB.exe`.
Start Le Mans Ultimate.
Run `LMUFFB.exe`.

4. Configure
------------
In LMUFFB, select your physical wheel from the "FFB Device" dropdown.
In Game, bind steering to the vJoy device.
Adjust settings in LMUFFB to taste.


TROUBLESHOOTING
================================================================================

Problem: "vJoyInterface.dll not found" error
Solution:
  - Ensure vJoyInterface.dll is in the same folder as LMUFFB.exe
  - If missing, copy from: C:\Program Files\vJoy\SDK\lib\amd64\ (64-bit)
                        or C:\Program Files\vJoy\SDK\lib\ (32-bit)

Problem: "Could not open file mapping object" message
Solution:
  - Start Le Mans Ultimate BEFORE running LMUFFB
  - Verify the rF2 Shared Memory Plugin is installed and enabled
  - Check Settings > Plugins in Le Mans Ultimate
